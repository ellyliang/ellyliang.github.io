译文：确保不在javascript中增加全局作用域2014-09-09

写javascript代码最好的做法是避免把对象放到全局作用域中。有几个很好的理由来解释全局变量会增加耦合（globals add coupling），例如，分开不同库的时候，很容易破坏彼此。事实上，通常程序的规则是避免增加全局作用域。遗憾的是，javascript增加对象到全局作用域中是非常容易的事情。参考以下这段代码：

function sayHello(person)
{
    message = "Hello, " + person;
    $("#message").text(message);
}

sayHello('@ardalis');

Javascript笔记之单例模式2014-07-11

什么是单例模式？

利用私有构造函数来保证只有一个实例，使不同的地方调用同一个实例对象。

单例模式最简单的例子：

var Useless = {
     name: "useless",
     age: function() {
       console.log("secret!");
     }
};
Useless.age();
